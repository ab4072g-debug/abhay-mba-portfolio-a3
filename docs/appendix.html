# Quarto-Specific Design Recommendations (3–5)

1. **Use a professional theme + typographic polish:** Start with a clean Quarto HTML theme (e.g., a Bootstrap-based theme) and customize `theme:` plus `css:` for spacing, readable line-length, and consistent heading hierarchy across pages.
2. **Navbar tools for quick access:** Add a right-aligned navbar area with icons/links for [LinkedIn] and an email link to [Email]; include a prominent “Download PDF” button on the Resume/CV page.
3. **Listings for Projects/Case Studies:** Implement a Quarto `listing` to render projects as cards with filters/tags (e.g., Strategy, Finance, Ops) and sorting (e.g., newest/featured), improving scanning and discoverability.
4. **Callouts for highlights and recruiter cues:** Use Quarto callouts (e.g., “Key takeaway,” “Impact,” “Tools used”) in each case study to standardize how outcomes and methods are presented.
5. **Tabsets for Skills organization:** Use tabsets on Skills & Certifications to separate “Business,” “Analytics,” and “Communication,” keeping the page concise while still complete.
